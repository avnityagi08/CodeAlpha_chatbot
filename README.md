# Basic Chatbot

A simple rule-based chatbot built using Python. The chatbot responds to predefined user inputs and demonstrates the use of functions, loops, and conditional statements.

## Features

- Responds to user greetings
- Answers simple questions
- Handles unknown inputs
- Exits when the user types "bye"

## Concepts Used

- Functions
- While Loops
- If-Elif-Else Statements
- Input/Output

## Supported Commands

- hello
- how are you
- what is your name
- bye

## How to Run

1. Make sure Python is installed.
2. Open the project folder in VS Code.
3. Run the following command:

```bash
python chatbot.py
```
